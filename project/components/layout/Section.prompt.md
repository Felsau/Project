Editorial section primitives — every sidebar panel is a stack of these.

```jsx
<Section>
  <SectionHead title="ดัชนีพืชพรรณ" meta="ฤดูแล้ง 2569" />
  …content…
</Section>

<Collapsible title="สถิติราย อำเภอ" meta="50 เขต" defaultOpen>
  …content…
</Collapsible>
```

Titles are 11px uppercase with 0.14em tracking; the rule below is a 1px hairline.
