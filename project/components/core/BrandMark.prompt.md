Brand diamond mark, optionally with the GreenLens wordmark — use in headers, footers, and report mastheads.

```jsx
<BrandMark withWordmark size={14} />
```

- `size` — diamond size in px (default 14; topbar uses 14, landing hero 18)
- `withWordmark` — adds the bold tracked wordmark
- `label` — defaults to "GreenLens"
